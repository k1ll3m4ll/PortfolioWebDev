Entwicklungsumgebung und Einrichtung

Visual Studio Code ist ein plattformübergreifender Editor für die Entwicklung in JavaScript. Mit Node.js kann JavaScript auch serverseitig verwendet werden, was die Anwendungsmöglichkeiten erweitert.

Nach der Installation können die Versionen von Node.js und npm mit node -v und npm -v überprüft werden. Alternativen wie Atom und Brackets eignen sich besonders für Anfänger.





Variablen und Datentypen

JavaScript unterstützt die Deklaration mit let, const und var. Der Typ der Variablen ist flexibel, z. B. kann eine Zahl später in einen Text umgewandelt werden.

Primitive Datentypen sind number, string, boolean, null, undefined, während Arrays und Objekte komplexe Typen darstellen.









Kontrollstrukturen

JavaScript bietet if-else-Bedingungen und Schleifen wie for und while, um den Programmfluss zu steuern. Eine Besonderheit ist die automatische Typkonvertierung, z. B. wenn prompt() einen String zurückgibt, selbst wenn Zahlen eingegeben werden.





Funktionen

Funktionen können deklarativ oder als Pfeilfunktionen definiert werden. Pure Functions haben keine Seiteneffekte und liefern Ergebnisse nur basierend auf ihren Eingaben. Higher-Order Functions wie map und filter sind essenziell für funktionale Programmierung.
