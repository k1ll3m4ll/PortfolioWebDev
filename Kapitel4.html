PHP und MySQL Grundlagen

Die Vorlesung behandelt die Integration von PHP mit MySQL-Datenbanken für Webanwendungen. Der Fokus liegt auf der Speicherung von Formulardaten in einer Datenbank statt per E-Mail. MySQL ist ein relationales Datenbankmanagementsystem, das Daten in Tabellen organisiert. Diese Tabellen bestehen aus Zeilen (Datensätzen) und Spalten (Feldern). Für die Interaktion mit MySQL wird die Abfragesprache SQL verwendet. Die Grundoperationen umfassen CREATE (Tabellen erstellen), INSERT (Daten einfügen) und SELECT (Daten abfragen). MySQL wird typischerweise parallel zum Webserver betrieben und über spezielle Client-Tools wie phpMyAdmin oder den mysql-Client verwaltet.

Datenbankverbindung mit PHP

PHP bietet spezielle Funktionen zur Interaktion mit MySQL-Datenbanken. Die wichtigsten sind mysqli_connect() (Verbindungsaufbau), mysqli_query() (Abfrageausführung) und mysqli_close() (Verbindungsschließung). Für die Verbindung benötigt man Serveradresse (meist 'localhost'), Benutzername, Passwort und Datenbankname. Die Funktion die() wird verwendet, um bei Fehlern die Ausführung abzubrechen. SQL-Abfragen werden in PHP als Strings aufgebaut und an mysqli_query() übergeben. Es ist wichtig, Datenbankverbindungen explizit zu schließen, um Ressourcen freizugeben.

Tabellen erstellen und Daten einfügen

Vor der Nutzung einer Datenbank müssen Tabellen mit definierten Spalten erstellt werden. Die CREATE TABLE-Anweisung spezifiziert Spaltennamen und Datentypen (z.B. VARCHAR für Text). Daten werden mit INSERT-Anweisungen hinzugefügt, wobei Werte in der Reihenfolge der Spalten oder mit expliziter Spaltenangabe eingefügt werden können. Formulardaten aus PHP werden über die $_POST-Superglobale in die INSERT-Anweisung integriert. Bei Textwerten müssen Anführungszeichen beachtet werden. Die erfolgreiche Ausführung von Abfragen kann mit SELECT * FROM tabellenname überprüft werden.

Daten abfragen mit SELECT

Die SELECT-Anweisung ermöglicht das Abrufen von Daten aus Tabellen. Mit * können alle Spalten ausgewählt werden, oder es können spezifische Spalten benannt werden. Die WHERE-Klausel filtert Ergebnisse nach bestimmten Kriterien (z.B. WHERE spalte = 'wert'). Dies ist besonders nützlich, um gezielt nach bestimmten Datensätzen zu suchen, wie im Beispiel der Entführungsberichte mit Hinweisen auf den Hund Fang. Abfragen können direkt in MySQL-Tools oder über PHP-Skripte ausgeführt werden. Die Ergebnisse werden tabellarisch zurückgegeben und können in PHP weiterverarbeitet werden.

MySQL-Werkzeuge

Für die Arbeit mit MySQL stehen verschiedene Werkzeuge zur Verfügung. Der mysql-Client ist ein Kommandozeilenprogramm für direkte SQL-Eingabe. phpMyAdmin bietet eine webbasierte grafische Oberfläche zur Verwaltung von Datenbanken. Beide Tools ermöglichen die Ausführung von SQL-Anweisungen, wobei phpMyAdmin zusätzliche visuelle Hilfen bietet. In der Entwicklungsumgebung XAMPP sind beide Tools typischerweise vorinstalliert. Für die Entwicklung ist es wichtig, zwischen lokalen und Produktions-Servern zu unterscheiden, da die Verbindungsparameter variieren können.

Sicherheitsaspekte

Bei der Datenbankprogrammierung sind Sicherheitsaspekte zu beachten. Datenbankzugänge sollten durch Benutzernamen und starke Passwörter geschützt sein. Formulardaten sollten vor der Verarbeitung validiert werden, um SQL-Injection-Angriffe zu verhindern. Die Vorlesung erwähnt, dass diese Themen in späteren Kapiteln vertieft werden. Grundsätzlich gilt: Niemals Benutzereingaben direkt in SQL-Abfragen einfügen ohne vorherige Prüfung und Escaping. PHP bietet dafür spezielle Funktionen wie mysqli_real_escape_string() oder Prepared Statements.
